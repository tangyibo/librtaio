librtaio
=========
  一个基于boost.asio实现的服务器端与客户端通信的网络协议。

**项目地址**

- github: https://github.com/tangyibo/librtaio

- gitee: https://gitee.com/inrgihc/librtaio

依赖
-----

- boost.asio库的使用;
 
目标
-----

- socket stream的解析;
- tcp protocol自定义协议;
- TCP分包与粘包;
 
编译
-----

```
git clone https://github.com/tangyibo/librtaio.git
cd librtaio
make clean
make all
```
